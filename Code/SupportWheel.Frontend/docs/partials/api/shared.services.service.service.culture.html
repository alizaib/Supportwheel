culture
service in module shared.services.service

Description

This service is responsable to get and set current culture for the application

Dependencies

  • $localStorage
    • Source: ngStorage
  • $mdDialog
    • Source: angular-material
  • $translate
    • Source: angular-translate

Methods

  • getCulture()

    Responsable to get the current culture selected in the application It reads from localstorage in the browser.

    Returns
    object

    It returns an object with the properties { culture (string) , name (string) }

  • init()

    It inits the service by loading the current set language from the localstorage. If no language is available it show a modal for the user to choose one. This modal cannot be closed or canceled.

  • load()

    It returns the collection 'culture' with its value set with the current culture.

  • onCultureChange(callback)

    Responsable to get the current culture selected in the application It reads from localstorage in the browser.

    Parameters
    ParamTypeDetails
    callbackfunction

    expect a callback function that will be called with the new culture set

    Returns
    function

    It returns an unsubscribe function that should be used on $destroy event in the controller

  • setCulture(culture)

    Responsable to set a new culture for the application. It saves the result in the localstorage of the browser

    Parameters
    ParamTypeDetails
    culturestring

    expect a culture in the format 'pt-BR'
