controller in module components.cultura.culturaSelect

Description

It has the logic behind the component

Methods

    This is a private function that is called when the controller is initialized and everytime an item or culture is changed

  • selectedItemChange()

    Everytime a user changes the selected value this item check if the value is different from the one initial set and if are, update the parameter 'id'. It also sets error messages based if the component is required or not
