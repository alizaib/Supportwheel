swEngineerForm
directive in module components.engineer.engineer.form

Description

swEngineerForm - Component

This component is responsible to render a form with all values of an engineer, so it can be updated or deleted.

Usage

as element:
<sw-engineer-form
       [id="{int}"]>
</sw-engineer-form>

Parameters

ParamTypeDetails
id
(optional)
int

Id of an Engineer to be loaded
