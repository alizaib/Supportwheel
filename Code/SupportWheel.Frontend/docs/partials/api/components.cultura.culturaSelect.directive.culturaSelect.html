culturaSelect
directive in module components.cultura.culturaSelect

Description

This component will render a select list with all values of 'cultura' and control the current culture set. The selected item will be available throught the parameterer 'id'

Usage

as element:
<cultura-select
       [id="{int}"]
       [required]>
</cultura-select>

Parameters

ParamTypeDetails
id
(optional)
int

The Id of the item that is selected or should be selected. It automatically maps to intIdCultura

required
(optional)
boolean

Sets if the controller is required or not when submitting a form
